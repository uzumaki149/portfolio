# Personal Portfolio Website

This is my first personal portfolio website built as part of my learning journey in web development.  
The project focuses on creating a clean, responsive, and interactive website using core front-end technologies.

## Overview
The portfolio demonstrates fundamental web development concepts, including layout structure, styling, dynamic content rendering, and basic user interaction. It is designed to be simple, readable, and scalable as more projects are added in the future.

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub Pages

## Features
- Responsive layout for mobile and desktop
- Dynamic project rendering using JavaScript
- Dark mode toggle
- Modal window for project details
- Deployed using GitHub Pages
